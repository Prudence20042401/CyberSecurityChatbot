# CyberSecurityChatbot

## Description
This is a chatbot built in C#.  
It helps people learn how to stay safe online.

## Features
- Plays a voice greeting  
- Shows a title on screen  
- Lets the user type questions  
- Gives answers about cybersecurity  

## How to Run
1. Open the project in Visual Studio  
2. Build the project  
3. Run the program  

## Tools Used
- C#
- Visual Studio
- GitHub
